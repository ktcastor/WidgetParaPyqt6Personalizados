# WidgetParaPyqt6Personalizados

Estos son los widgets que puedes usar para tus aplicaciones creadas con pyqt6 en python

CassieCarrusel
Widget que crea un carrusel de imágenes y muestra su vista previa

CassieImageLabel
WIdget que muestra las imágenes en el carrusel

CassieButton
Bóton personalizado para pyqt6

CassieSearch
Widget para crear un buscador con elemntos predefinidos pasados en forma de json o lista
